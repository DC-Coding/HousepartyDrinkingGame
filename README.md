# Houseparty Drinking Game

# How to add code

## Clone the project
```
git clone https://github.com/DC-Coding/HousepartyDrinkingGame.git
```

## Checkout your own branch
```
git checkout <Name>
```

## Commit changes
```
git add --all
git commit -m "Your changes"
```

## Push to the server
```
git push -u origin <Name>
```
